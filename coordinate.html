Step 1: Coordinate System

Let's first get familiarized with how to draw lines on the screen. For now, think of pad.makeLine as a magic command that we can write to draw lines on the screen. To instruct this magic function to draw lines, we need to communicate the starting and ending point of the line.

Remember the graphs you drew in math class, where each point has an x and a y coordinate? It's the same deal here. We will call the starting point's coordinates x1 and y1, and the ending point's coordinates x2 and y2. Drawing a line is as simple as typing pad.makeLine(x1,y1,x2,y2), changing the symbols to actual numbers like pad.makeLine(0,0,100,100).

The dimensions of the drawing canvas are 400 X 400, so all coordinate values need to be positive and less than 400. The top-right of the canvas is (0, 0) and bottom-right is (400, 400).
