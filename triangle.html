Step 2: Triangles

Let us now see how we can use our knowledge of line-drawing to draw a triangle. Since a triangle is just three lines, if we have three points (x1, y1), (x2, y2) and (x3, y3), we could draw a line between each of the points to draw a triangle. Simple, right?

Below, you will notice the first line of code reads let x1 = 100. The command let creates something called a variable. You can think of a variable as a box that stores a value inside of it - whenever we open the box x1, we will get out the value 100. For now, we use these boxes just so we can have a consistent way of referring to the same value multiple times.
